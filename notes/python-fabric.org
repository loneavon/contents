* 远程 copy 文件
可利用 put，put 的第一参数是本地文件路径，第二参数是远程文件路径。如下：
#+BEGIN_SRC python
from fabric.api import run, put 

def vimrc():                                                                                           
  run('rm -rf ~/.vimrc ~/.vim')                                                                        
  put('~/.vimrc', '~/')                                                                                
  put('~/.vim', '~/')  
#+END_SRC

* 并行
#+BEGIN_SRC python
from fabric.api import run, parallel

@parallel                                                                                              
def install_vim():                                                                                     
  run('jumbo install vim')  
#+END_SRC
